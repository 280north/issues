How To File Helpful Bug Reports

Things to include with your bug report:

  • What version are you running? - It's possible this bug was already fixed but hasn't been included with the latest release of the software. Including the version number will help the developers track down the bug more easily.
  • What did you do to cause the issue? - How can the developers reproduce this issue? If you don't know what causes the bug, give as much detail as you can about what you were doing when you noticed it.
  • What should have happened? - When you saw the bug what did you expect to happen?
  • What actually happened? - When the bug occurred what happened to the software? Be sure to include any error messages you found.
  • What operating system and browser are you running? What version? - It's possible the bug you've run across only affects certain operating systems or browsers. You should include all of that information with your bug report!
